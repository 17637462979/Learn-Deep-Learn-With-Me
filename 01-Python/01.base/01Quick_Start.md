
## 程序输出
- 将一个字符串赋值给a
- 用print显示变量内容


```python
a = 'Hello World'
```


```python
print(a)
```

    Hello World


## 程序输入
- Python3中input()默认数据为字符串


```python
age = input('enter your age:')
```

    enter your age:22


### 打印age


```python
print(age)
```

    22


### 打印age的类型
- age的类型为str(字符串)


```python
print(type(age))
```

    <class 'str'>

## 比较运算符
- < <= 
- ==:等于
- !=:不等于
- 比较运算根据表达式的值的真假返回布尔值


```python
2 < 4
```




    True




```python
2 == 4
```




    False




```python
3 != 4
```




    True
